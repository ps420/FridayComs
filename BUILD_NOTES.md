# Build Instructions for FridayComs (Windows)

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
   ```bat
   dotnet --version
   ```

2. Restore NuGet packages.
   ```bat
   dotnet restore
   ```

3. Build the project.
   ```bat
   dotnet build
   ```

4. Run the application.
   ```bat
   dotnet run
   ```

Feel free to adjust the `dotnet` commands for your specific .NET target framework or solution layout.
